# Notebooks Directory

This directory contains Jupyter notebooks for experimentation, analysis, and demonstrations.

## Recommended Notebooks

- `01_data_exploration.ipynb`: Explore and visualize the dataset
- `02_preprocessing.ipynb`: Test preprocessing techniques
- `03_plate_detection.ipynb`: Develop and test plate detection models
- `04_ocr_recognition.ipynb`: Develop and test OCR models
- `05_full_pipeline.ipynb`: End-to-end pipeline demonstration
- `06_model_evaluation.ipynb`: Model performance analysis

## Usage

```bash
jupyter notebook
```

Navigate to this directory to access the notebooks.
